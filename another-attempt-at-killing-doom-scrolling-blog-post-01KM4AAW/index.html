Another attempt at killing doom scrolling

A personal reflection on setting up VPN-based internet blocking to stop doom scrolling and reclaim attention.

This post was published on March 20, 2026 by Avinash and was last edited on April 3, 2026.

Source: This post originated from my journal entry on March 20, 2026.

This is my another new attempt at blocking my internet access and doom scrolling throughout the internet. Let’s see how it goes again.

I have set up the VPN with NextDNS configured in it, so that blocks the websites that I don’t want to access. It triggers every few minutes, so even though I disable the VPN to access the broader internet, it will trigger itself and enable the NextDNS VPN.

So yeah, let’s see how useful it is.

2026-04-03

This did not work again. I think what will work is training my mind to actually self-limit the doom scrolling instead of making or using external things to stop it. So basically, I have to be self-aware of the things I’m doing and when I’m doing them. This is a really big problem because I have a really huge exam coming up, and I’m not preparing the way I should be. This is really a huge, huge, huge concern.
